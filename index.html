Como Jogar

  • A / ← mover para Esquerda
  • D / → mover para Direita
  • S / ↓ / ESPAÇO cavar bloco abaixo
  • ESPAÇO + A/D cavar lateral (para o lado pressionado)
  • E interagir / abrir Baú
  • U abrir menu de upgrades (na superfície <= 5 m)
  • Botão SUPERFÍCIE teleporta você de volta (use se ficar preso)
  • ESC fechar cartões / pausar

Aprofunde-se para encontrar minérios. Quanto mais fundo, mais raros. Ao achar um Baú do Tesouro, abra-o para receber um Card de Arqueologia com curiosidades reais sobre o estudo do passado humano.
